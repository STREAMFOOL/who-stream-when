# Product Overview

**Who Live When** is a streaming schedule tracker that helps users follow content creators across multiple platforms (YouTube, Twitch, Kick).

## Core Features

- **Multi-platform tracking**: Aggregates live status from YouTube, Twitch, and Kick
- **Activity heatmaps**: Generates probability-based predictions of when streamers go live based on historical data
- **TV Programme view**: Weekly calendar showing predicted streaming times for followed streamers
- **Google OAuth authentication**: Users log in to follow streamers and view personalized schedules
- **Live status monitoring**: Real-time tracking of who is currently streaming

## User Flow

1. Guests browse the home page showing most viewed streamers and their live status
2. Users log in with Google to follow specific streamers
3. Dashboard displays a personalized weekly schedule based on activity patterns
4. Search functionality helps discover new streamers across platforms
5. Individual streamer pages show live status, activity heatmaps, and platform links
